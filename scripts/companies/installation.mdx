First of all make sure you've installed and started all the required dependencies **before** the mega_companies resource in your *resource.cfg* file and that you ran the .sql file provided for creating the needed database tables.

## Dependencies (must be started before)

- VORP Inventory
- VORP Core
- mega_notify
- mega_nplayerselector
- mega_inputs
- mega_messagebuilder
- uiprompt (kibook)
